Descripción del Proyecto

Una de las áreas más atendidas ha sido el soporte de equipo informático. Al haber laborado en un entorno educativo, el mantenimiento preventivo y el cambio de piezas de hardware para mejorar el rendimiento de los sistemas era bastante recurrentes. Junto a esto, se sumaban equipos de proyección, monitoreo y audio, de los cuales se realizaban cotizaciones con diferentes proveedores para gestionar de forma eficiente la adquisición de repuestos críticos, tales como lámparas para los proyectores.

Según las capacidades y la arquitectura del hardware, se gestionaban recomendaciones personalizadas de software y despliegues de sistemas operativos, utilizando entornos Windows o distribuciones optimizadas como Linux Mint para estaciones de trabajo de recursos limitados, aplicando de manera simultánea directivas de seguridad informática esenciales.

Más allá del marco institucional, clientes particulares confiaban de manera regular sus equipos informáticos de gama baja y media o laptops para optimizaciones, limpiezas físicas exhaustivas y actualizaciones de componentes de hardware (SSD, RAM), además de la resolución de consultas técnicas y soporte a funcionalidades lógicas del sistema.
